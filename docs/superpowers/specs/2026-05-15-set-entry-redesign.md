# 训练组录入重构设计

## 总览

重构弹窗内训练组录入区域：五列布局、组类型切换、自定义数字键盘。

## 1. 数据结构变更

每组扩展两个字段：

```js
// 旧格式 → 新格式
{ reps: 8, weight: 60 } → { reps: 8, weight: 60, type: 'normal', done: false }
```

- `type`: `'normal'` | `'warmup'` | `'dropset'`，默认 `'normal'`
- `done`: `boolean`，默认 `false`

迁移逻辑：读取数据时自动补全缺失字段。

## 2. 五列表格布局

替换现有 `.set-detail-row` 的 flex 布局为 CSS Grid 五列：

| 列 | 宽度 | 内容 | 交互 |
|---|---|---|---|
| 组 | ~18% | 可点击标签 | 点击弹出类型菜单 |
| 上一次 | ~20% | 只读灰色文字 | 无 |
| 重量 | ~24% | 数值单元格 | 点击打开数字键盘 |
| 次数 | ~20% | 数值单元格 | 点击打开数字键盘 |
| ✓ | ~10% | 打钩框 | 点击切换完成状态 |

列头在组数控制下方显示。

## 3. 组类型切换

- 「正式组」→ 显示 `第x组`（x 为正式组序号），蓝色标签
- 「热身组」→ 显示 `热`，黄色标签
- 「递减组」→ 显示 `减`，粉色标签

点击标签弹出小菜单（absolute定位），选类型后更新标签和序号。

序号逻辑：遍历当前所有组，只对 `type === 'normal'` 的组递增计数。

## 4. 自定义数字键盘

全宽底部弹出面板，CSS Grid 4 列布局，键盘宽度 = sheet 宽度。

```
第一行:  1    2    3    ⌫
第二行:  4    5    6    +
第三行:  7    8    9    −
第四行:  .    0    确认(span 2)
```

- 焦点提示行显示「正在输入：重量 (±0.25 kg)」或「次数 (±1 次)」
- ± 按键：重量焦点 → 步长 0.25，次数焦点 → 步长 1
- 小数点：只有重量模式可用
- 确认/回车 → 收起键盘

阻止系统键盘：重量/次数用 `<div>` + `tabindex` 模拟输入，不用 `<input>`。
